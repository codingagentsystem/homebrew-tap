# Homebrew Tap for CAS

This is the official Homebrew tap for [CAS - Coding Agent System](https://github.com/codingagentsystem/cas).

## Installation

```bash
brew tap codingagentsystem/tap
brew install cas
```

Or install directly:

```bash
brew install codingagentsystem/tap/cas
```

## Updating

```bash
brew update
brew upgrade cas
```

## Supported Platforms

- macOS (Apple Silicon / ARM64)
- Linux (x86_64)
